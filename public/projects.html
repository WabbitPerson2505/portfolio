Projects

Lists of projects:

  1. Dataset prediction using java.

    Previous project from iti1121 class. Coded in java, the aim of the program is to predict outcomes of a set of data provided using problabilities. The project can only be run on datasets of csv formats.

  2. Hotel database systems.

    Previous project from csi2132. Conception of a simulated database to run as the backend for a hotel reservation system. The purpose of the database is perform the operation of a typical reservation system such as performing reservations, loading reservations, adding hotels into the system. This also includes the creation of reports and such. This project uses PostgreSQL as the database and pgadmin as the database management tool.

  3. Yatzy

    Project from csi3140 (Web). Implementation in html of a Yatzy game. Note this is Yatzy and not Yahtzee, Yatzy is a variation on Yahtzee and has some different rules from it. The game can only run 1 round and not multiple games. This game was made in both a javascript version for local and php for servers.

  4. Hospital triage app

    Project from csi3140 (Web). Implementation of a simple web app that manages the triage of a hospital. Staff can use the web app to see the queue of patients in need of treatment while patients can login to see an approximate wait time to their appointment. This web app use postgresql for the database and php for the server interactions.
